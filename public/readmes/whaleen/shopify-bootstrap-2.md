# Shopify Bootstrap 2 Theme

A Shopify theme built with Twitter Bootstrap 2, designed for stores with complex product variants and custom requirements.

## Features

- **Bootstrap 2 Framework**: Responsive grid system and UI components
- **Advanced Product Variants**: Custom Liquid markup for handling complex product options
- **Responsive Design**: Mobile-first approach with Bootstrap's responsive utilities
- **Custom Liquid Templates**: Optimized for stores requiring detailed product customization
- **Production-Ready**: Battle-tested in retail environments

## Project Type

Legacy Shopify theme template (Bootstrap 2 era) - suitable for reference or as a foundation for modern Bootstrap upgrades.

## Technologies Used

- Shopify Liquid templating
- Twitter Bootstrap 2
- HTML5/CSS3
- JavaScript/jQuery
- Responsive design patterns

## Template Status

- **Clean Template**: All client-specific pages and assets have been removed
- **Ready for Customization**: Generic foundation suitable for any retail store
- **Legacy Framework**: Bootstrap 2 era - consider upgrading to modern Bootstrap for new projects

## Structure

- `shopify_theme/` - Main theme files
- `templates/` - Liquid templates for different page types
- `snippets/` - Reusable Liquid components
- `assets/` - CSS, JS, and image files
- `pages/` - Custom page templates (cleaned - add your own as needed)
